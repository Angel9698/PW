Case Study #4 | Deep Dive!

box

This project focuses on improving the usability and functionality of a small business website that was built with a multipurpose template. Through a structured heuristic evaluation, I identified critical points impacting user trust, clarity, and task completion, and proposed targeted design improvements.

Problem

The website suffered from a combination of usability, content, and visual design issues that created a poor first impression and hindered the users ability to complete tasks and achieve their goals, such as:

  • Confusing navigation and deceptive interactions
  • Unnecessary sections and clutter
  • Poor grammer and punctuation

Methodology

I evaluated the usability of the site using Nielson's Usability Heuristics and categorized issues by severity:

  • Critical (urgent usability failures)
  • Moderate (confusing/inefficient interactions)
  • Minor (aesthetics or in need of polish)

Key Findings

Weak First Impression & Redundant Actions

  • Duplicated buttons and unnecessary content
  • Grammar and punctuation errors
  • Poor logo placement

    • Deceptive Interactions

    • Hover states implied clickability where there is none
    • Buttons lead to unexpected destinations (third-party site)

      • Poor Content Hierarchy & Spacing

      • Lack of scannability
      • Poor visual hierarchy with headings and sub-headings
      • Inconsistent layouts conflict with content presentation

        • User Goals Not Supported

        • Unclear contact form labels
        • Absence of the shop's location
        • Lack of button functionality to support users goals

          • Poor Mobile Experience

          • Hamburger menu is empty when clicked

            • Design Solutions

            Content Simplification

            • Remove/re-purpose duplicated buttons/content
            • Ensure all elements have a clear, meaningful purpose

              • Fix Unexpected Interactions

              • Disable decpective hover states
              • Ensure all clickable elements behave within user expectations

                • Content Optimization

                • Break down blocks of text
                • Introduce sub-sections
                • Use clearer section titles

                  • Mobile

                  • Remove useless hamburger menu

                    • General Usability

                    • Use standard form labels
                    • Reorganize layouts for better flow
                    • Add functional links that supports user goals (location, phone-call)

                      • Outcomes

                      User Impact

                      • Faster task completion and access to key information
                      • Reduced cognitive load and content aligns with user expectations

                        • Business Impact

                        • Improved professionalism and credibility with first-impressions
                        • Reduced abandonment rates
                        • Higher action potential
